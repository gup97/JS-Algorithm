### 처음 푼 과정(풀 때 생각)
        간단한문제네
        모듈러연산->sort
        함수형으로 해봐야지

```JS
function solution(arr, divisor) {
    const result =arr.filter(el=>el%divisor===0).sort((a,b)=>a-b)
    return result.length?result:[-1];
}
```
