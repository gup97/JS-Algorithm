[두 정수사이의합](https://programmers.co.kr/learn/courses/30/lessons/12912?language=javascript)

```JS
function solution(a, b) {
    var answer = 0;
    const [min,max] = a<b?[a,b]:[b,a];
    for(let i =min;i<=max;i++){
        answer +=i
    }
    return a===b?a:answer;
}
```
