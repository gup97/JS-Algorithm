## 처음풀때

    약수구하는 문제

```JS
function solution(n) {
    var answer = 0;
    for(let i=1;i*i<=n;i++){
        if(n===(i*i)){
            answer+=i
        }
        else if(n%i===0){
            answer+=i;
            answer+=n/i;
        }
    }
    return answer;
}

```
다 더한후 빼는 게 더 깔끔해보인다  

```JS
function solution(n) {
    var answer = 0;
    let i;
    for (i = 1; i <= Math.sqrt(n); i++){
        if (!(n%i)) {
            answer += (i+n/i);
        }
    }
    i--;
    return (i === n/i) ? answer-i : answer;
}
```
